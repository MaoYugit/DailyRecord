| 属性            | 描述                                     |
| --------------- | ---------------------------------------- |
| border-collapse | 设置表格的边框是否被折叠成一个单一的边框 |
| border-spacing  | 设置相邻单元格的边框间的距离             |
| caption-side    | 指定表格标题的位置                       |
| empty-cells     | 设置是否显示表格中的空单元格             |
